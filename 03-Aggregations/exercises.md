# SQL Exercises

## Instructions
- Use the provided sample tables or create your own.
- Write each solution in a separate `.sql` file.
- Try to solve without checking references.
- Focus on accuracy and readability.
- Add more questions as you learn.

---

## Basic Exercises

1. Display all records from a table.
2. Select only specific columns from a table.
3. Filter records where a numeric column is greater than a given value.
4. Filter records based on text matching.
5. Sort records in ascending and descending order.
6. Find records within a given range.
7. Count the total number of records in a table.
8. Find distinct values in a column.
9. Limit the number of rows returned.
10. Rename columns using aliases.

---

## Aggregate Exercises

11. Find the total sum of a numeric column.
12. Calculate the average value of a column.
13. Find the minimum and maximum value.
14. Count records grouped by a category.
15. Find total values per category.
16. Filter grouped data using HAVING.
17. Find categories with highest totals.
18. Calculate overall statistics using multiple aggregates.

---

## Joins Exercises

Assume tables:

```
customers(customer_id, name, city)
orders(order_id, customer_id, amount)
```

19. Display all customers with their orders.
20. Show customers who have not placed any orders.
21. Calculate total order amount for each customer.
22. Find customers with total orders greater than a given value.
23. Count number of orders per customer.

---

## Subqueries Exercises

24. Find records greater than the average value.
25. Display the second highest value.
26. Find records belonging to the same category as a given record.
27. Find records that do not exist in another table.
28. Find top N values using subquery.

## Window Functions Exercises

29. Rank records based on a numeric column.
30. Calculate running totals.
31. Find top record in each category.
32. Compare values using LAG or LEAD.
